# SimpleToken-Educational-
This project is an minimal ERC20 token application created with Solidity. It was prepared as a study to demonstrate an understanding of token logic and smart contract fundamentals.

# Features
- Token data (symbol, name, decimal)
- Balance tracking using mapping
- Token transfer functionality
- Owner-only minting
- Token burning functionality
